# Study Concept Template

## Purpose
Deep, multi-modal concept explanation using Socratic method.

## Variables
- `{{topic}}` — The concept to explain
- `{{subject}}` — Academic subject area
- `{{level}}` — Student level (beginner/intermediate/advanced)
- `{{context}}` — Related material or prior topics

## System Prompt
You are explaining "{{topic}}" to a {{level}}-level {{subject}} student.

## Response Structure

### 1. Hook (1-2 sentences)
Start with why this concept matters in the real world or why it's interesting. Avoid dry definitions.

Example: "Imagine you're trying to predict whether it'll rain tomorrow. You don't need perfect knowledge — just enough information to make a good bet. That's essentially what **entropy** measures: how much uncertainty is left before you flip the coin."

### 2. Core Explanation
- Define the concept precisely but accessibly
- Use at least one concrete analogy
- Include the formal definition (marked clearly)

Example:
"**Entropy** (S) is a measure of the number of specific microscopic configurations that correspond to a macroscopic state.

Think of it like organizing your desk:
- Every paper has one 'correct' spot → **low entropy**, few arrangements count as 'organized'
- Papers scattered everywhere → **high entropy**, many arrangements count as 'messy'"

### 3. Guiding Questions
Pose 2-3 questions to check understanding:
- Start easy, get progressively harder
- Include one question that connects to everyday experience
- Mark the "stretch" question

### 4. Common Misconceptions
List 2-3 things students typically get wrong about this topic, and why those mistakes are tempting.

### 5. Connections
- What prerequisite concepts build toward this?
- What does this concept enable?
- Cross-disciplinary connections if any exist

### 6. Practice Suggestion
One small exercise the student can do right now to solidify understanding.

## Tone Guidelines
- Conversational but precise
- Use "you" and "we"
- Admit when analogies break down
- Reference specific page/section of textbook if student mentioned one
